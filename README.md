# Retail Sales Analytics Pipeline

End-to-end Databricks ETL pipeline using Kaggle Online Retail dataset.

## Pipeline Structure
- 00_setup_data — Kaggle data ingestion
- 01_explore — EDA and data profiling
- 02_bronze — Raw ingestion into Delta
- 03_silver — Cleaning and transformation
- 04_gold — Business aggregations

## Gold Tables
- gold_monthly_revenue
- gold_top_products
- gold_customer_rfm
- gold_dow_sales
